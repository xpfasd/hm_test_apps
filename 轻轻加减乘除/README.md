# 数学小天才 - 加减乘除练习应用

## 项目介绍
这是一个专为小学低年级学生设计的数学练习应用，通过丰富的动画演示帮助孩子理解加减乘除的运算过程。

## 主要特性

### ✨ 核心功能
- **四大运算模块**：加法、减法、乘法、除法
- **每种运算20个子类别**：从基础到进阶，循序渐进
- **每个类别100道题目**：充分练习，巩固知识
- **动画演示**：每道题都有生动的动画展示运算过程
- **离线使用**：支持PWA，可离线使用
- **进度保存**：自动保存学习进度

### 🎮 游戏化功能
- **用户系统**：个人档案、头像、昵称设置
- **等级系统**：通过答题获得经验值，提升等级
- **成就徽章**：20种成就等你解锁
- **每日任务**：完成任务获得额外奖励
- **排行榜**：查看学习排名，激发竞争意识

### 📚 学习辅助
- **错题本**：自动收集错题，随时复习
- **学习统计**：详细的学习数据分析
- **挑战模式**：限时挑战、生存模式
- **进度追踪**：可视化学习进度

### 🎯 运算类别

#### 加法练习
- 基础范围：1-5、10以内、20以内、50以内、100以内
- 特殊练习：整十数、进位加法、相同数相加、加1/2/5/10练习
- 高级练习：三数连加、填空练习、应用题、钱币计算等

#### 减法练习
- 基础范围：5以内、10以内、20以内、50以内、100以内
- 特殊练习：整十数、退位减法、从10减、减1/2/5/10练习
- 高级练习：连续减法、填空练习、应用题、找零钱等

#### 乘法练习
- 乘法口诀：1-9的完整乘法口诀表
- 特殊练习：乘以0/1/10、平方数、翻倍练习
- 高级练习：填空练习、应用题、阵列理解等

#### 除法练习
- 基础除法：除以1-9的练习
- 特殊练习：除以10、自除、求一半、整除练习
- 高级练习：填空练习、应用题、平均分配等

### 🎨 界面特色
- **儿童友好设计**：色彩鲜艳、图标生动
- **响应式布局**：完美适配手机和平板
- **动画效果**：
  - 加法：物品逐个累加动画
  - 减法：物品逐个消失动画
  - 乘法：分组展示动画
  - 除法：平均分配动画

## 使用方法

### 在线使用
1. 直接在浏览器中打开 `index.html` 文件
2. 选择想要练习的运算类型
3. 选择具体的练习类别
4. 输入答案并提交
5. 观看动画演示，理解运算过程

### 安装为手机应用
1. 在手机浏览器中打开应用
2. 点击浏览器菜单中的"添加到主屏幕"
3. 应用将作为独立APP安装到手机上
4. 支持离线使用

### 部署到服务器
1. 将所有文件上传到Web服务器
2. 确保服务器支持Service Worker（需要HTTPS）
3. 访问网站即可使用

## 技术栈
- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **动画**：CSS Animations + JavaScript动画
- **离线支持**：Service Worker + Cache API
- **数据存储**：LocalStorage
- **PWA支持**：Web App Manifest

## 图标生成
为了完整的PWA体验，需要生成应用图标：

1. 创建一个 512x512 的应用图标（可以使用数学相关的设计）
2. 使用在线工具生成各种尺寸：
   - 72x72, 96x96, 128x128, 144x144
   - 152x152, 192x192, 384x384, 512x512
3. 将图标文件放在 `icons/` 目录下

推荐的在线工具：
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/
