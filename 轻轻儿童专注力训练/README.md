# 🎯 儿童专注力训练项目

一个专为儿童设计的移动端专注力训练HTML应用，包含100个关卡和20种不同类型的项目。

## 📱 特点

- **移动端优先设计**：竖屏适配，适合手机和平板使用
- **儿童友好界面**：大色块、可爱的动物图标
- **丰富的交互体验**：点击音效、动画反馈、庆祝特效
- **渐进式难度**：每种项目类型包含5个由易到难的关卡
- **进度保存**：使用localStorage自动保存项目进度
- **星级评价系统**：根据完成时间和错误次数给予1-3星评价

## 📂 项目结构

```
.
├── index.html              # 主页面（关卡选择）
├── game1-level1.html       # 第1关：找相同（入门）
├── game1-level2.html       # 第2关：找2只相同
├── game1-level3.html       # 第3关：找3只相同
├── game1-level4.html       # 第4关：找4只相同（有错误限制）
├── game1-level5.html       # 第5关：找5只相同（挑战关）
├── css/
│   └── style.css           # 全局样式
├── js/
│   └── main.js             # 项目逻辑和进度管理
└── README.md               # 说明文档
```

## 🎮 已完成功能

### 第一种项目类型：找相同小动物 🦁

**项目玩法**：在多个动物中找出与目标相同的小动物

**5个难度等级**：
1. **第1关**：6个动物中找1只相同的
2. **第2关**：9个动物中找2只相同的
3. **第3关**：12个动物中找3只相同的
4. **第4关**：16个动物中找4只相同的（错误限制3次）
5. **第5关**：20个动物中找5只相同的（挑战关，错误限制3次）

### 核心功能

✅ **关卡管理系统**
- 关卡解锁机制（完成前一关才能解锁下一关）
- 进度自动保存
- 星级评价系统

✅ **交互动画**
- 点击反馈动画
- 正确/错误选择动画
- 弹窗动画效果
- 关卡完成时的庆祝特效（第5关）

✅ **音效系统**
- 点击音效
- 正确答案音效
- 错误答案音效
- 胜利音效

✅ **响应式设计**
- 适配不同尺寸的移动设备
- 禁止缩放和文本选择
- 触摸优化

## 🚀 使用方法

1. **直接打开**：在浏览器中打开 `index.html`
2. **本地服务器**（推荐）：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (需要安装 http-server)
   npx http-server
   ```
3. 访问 `http://localhost:8000`

## 🎨 设计特点

- **色彩丰富**：使用渐变色和高对比度配色
- **大图标**：适合儿童点击的大尺寸按钮和卡片
- **可爱动物**：使用emoji动物图标，无需额外资源
- **即时反馈**：每次操作都有视觉和听觉反馈

## 📊 星级评价标准

星级根据完成时间和错误次数计算：
- ⭐⭐⭐：快速完成且少错误
- ⭐⭐：中等速度或有些错误
- ⭐：完成时间较长或错误较多
